# Quickstart: 「如何呼叫」可發現性重設計

## 給成員（怎麼用）

1. 進**金鑰**頁 → 每把金鑰旁有「**如何使用這把金鑰**」。
2. 從下拉**選一個模型**（只列**這把金鑰能用**的模型）→ 立刻看到可複製的呼叫範例（平台端點 + `$TOKEN` 佔位 + 該模型的正確識別字）。把 `$TOKEN` 換成你建立金鑰時拿到的 token 即可。
3. 想接**工具**（Codex…）或自己**寫程式**？進**應用**頁——「工具整合」卡有各工具設定，「直接用 API / SDK」卡有 curl/Python/JS 範例。
4. 在儀表板、分配、模型頁也都看得到通往「如何使用 / 應用」的入口。

> 金鑰只會顯示一次；範例用 `$TOKEN` 佔位，不會再顯示你的金鑰明文。

## 給維護者（驗收重點）

- **不鑽詳情就找得到**：金鑰頁直接有「如何使用這把金鑰」（SC-001）。
- **下拉＝這把金鑰 scope**：切換金鑰，模型清單跟著變、不混入其他金鑰的模型（SC-003）。
- **範例對型**：選 embedding/OCR/即時字幕 → 範例端點正確對應（SC-002）。
- **單一來源**：範例都走 `ApiUsageExample`，改一處全處同步（SC-006，無重複文本）。
- **應用總站**：應用頁有「工具整合」+「直接用 API / SDK」兩類（SC-004）。
- **cross-link**：儀表板/分配/模型詳情都有入口（SC-005）。
- **零後端**：無新端點、無 migration；`ruff`/mypy 不受影響，前端 `tsc` + vitest + build 綠。
